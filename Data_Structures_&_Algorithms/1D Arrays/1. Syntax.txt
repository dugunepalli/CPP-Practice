One-dimensional arrays, also called as single arrays

Syantax is "Data_Type Array_Name[Array_Elements]"

Examples:
a. int Sunil[5]={1,2,3,4,5};
b. char Sunil[5]={'a','b','c','d};
c. int Sunil[]={1,2,3,4,5};    #Auto Update

Indexing: as refer using Example a
    1. Array indexing Starts with 0
    2. indexing to Access/Manipulte/Push data
        Sunil[0]=1
        Sunil[1]=2
        Sunil[2]=3.....
